# Inicio Rápido

Pon tu sistema de facturación en funcionamiento en menos de 5 minutos.

<Callout type="info">
**Prerrequisitos:** Necesitarás una cuenta de Commet y una clave API. [Regístrate aquí](https://demo.commet.co) si no tienes una.
</Callout>

<Steps>
<Step>

## Instala el SDK

<Tabs items={['npm', 'pnpm', 'yarn']}>
<Tab value="npm">
```bash
npm install commet
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add commet
```
</Tab>
<Tab value="yarn">
```bash
yarn add commet
```
</Tab>
</Tabs>

</Step>

<Step>

## Inicializa el cliente

Crea tu cliente Commet con tu clave API:

```typescript
import { Commet } from 'commet'

const commet = new Commet({
  apiKey: 'tu-clave-api',
  environment: 'sandbox' // Por defecto - seguro para pruebas
})
```

<Callout type="warn">
**Mantén tu clave API segura:** Nunca expongas tu clave API en código del lado del cliente. Usa variables de entorno.
</Callout>

</Step>

<Step>

## Envía tu primer evento de uso

Rastrea el uso para facturación en una línea:

```typescript
// Rastrea llamadas API para facturación basada en uso
await commet.usage.create({
  eventType: 'api_call',
  customerId: 'cus_A1b2C3d4E5f6G7h8I9j0K1l2',
  quantity: 1,
  timestamp: new Date().toISOString(),
})
```

</Step>

<Step>

## Gestiona licencias de asientos

Agrega o remueve asientos para facturación por suscripción:

```typescript
// Agregar 5 asientos de admin
await commet.seats.add({
  customerId: 'cus_A1b2C3d4E5f6G7h8I9j0K1l2',
  seatType: 'admin',
  quantity: 5,
})

// Remover 2 asientos
await commet.seats.remove({
  customerId: 'cus_A1b2C3d4E5f6G7h8I9j0K1l2', 
  seatType: 'admin',
  quantity: 2,
})
```

</Step>
</Steps>
