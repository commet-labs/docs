# Onboarding

En esta guía encontrarás los pasos a seguir si es la primera vez que ingresas a Commet y deseas probar sus funcionalidades base.

<Callout type="info">
Recuerda que para empezar a operar con Commet de manera real, deberás hablar con nuestro equipo para poder completar el onboarding legal y seleccionar un plan de nuestro pricing. Nuestro equipo te ayudará en todo el proceso.
</Callout>

<Steps>
<Step>

### 1. Crea una cuenta

Ve a https://sandbox.commet.co. Si es la primera vez que accedes, tendrás que completar algunos datos básicos sobre tu empresa.

</Step>

<Step>

### 2. Crea tus tasas de impuestos

Todos los productos de Commet deben tener un tipo de producto y una tasa de impuestos asociada. Esto nos permite calcular los impuestos directamente. Tendrás que crear al menos una categoría de producto y una tasa de impuestos.

<Callout type="info">
No te preocupes, siempre puedes marcar a tu cliente como exento para evitar calcular impuestos, y estos no aparecerán en la factura.
</Callout>

</Step>

<Step>

### 3. Crea o importa un cliente

Como primer paso, necesitarás crear un cliente desde la pestaña Customer. Allí definirás sus datos de facturación, moneda y condiciones.

</Step>

<Step>

### 4. Crea tipos de eventos y métricas

Necesitas crear tipos de eventos o tipos de asientos, que definen los tipos de uso que quieres rastrear. Estos eventos deben agruparse luego en métricas, ya que las métricas son lo que se asocia con los productos más adelante en el proceso.

<Callout type="info">
Más adelante, podrás enviar eventos vía API para probar con datos reales, pero para este onboarding puedes dejar esta sección vacía—el modo sandbox te permite simular todo con datos de prueba. El objetivo de este onboarding es mostrarte el flujo completo, hasta generar una factura final.
</Callout>

</Step>

<Step>

### 5. Crea tus productos

Desde la pestaña de productos, crea tus productos, ya sean de una vez, recurrentes, fijos o vinculados a alguna métrica creada anteriormente.

</Step>

<Step>

### 6. Crea un Agreement

Un Agreements es donde se reúnen todas las condiciones acordadas con tu cliente. Cada agreement puede tener diferentes fases y determinará Qué y Cómo cobraremos a nuestros clientes.

</Step>

<Step>

### 7. Generación de facturas

En esta etapa, podrás simular facturas para ver exactamente cómo se vería una factura para este acuerdo. Cuando llegue la fecha de facturación establecida en el acuerdo, se generarán facturas usando datos reales, si están disponibles.

<Callout type="warning">
No podrás enviarlos ni descargarlos si no tienes una cuenta activada por nuestro equipo.
</Callout>

</Step>

<Step>

### 8. Próximos pasos

En este punto, tienes dos opciones:

**Opción A: Obtener una cuenta de producción**
[Reserva un slot con nuestro equipo](https://cal.com/guido-irigoyen/onboarding-meeting) para que podamos ayudarte a encontrar el mejor plan para ti. Nuestro equipo se encargará de la migración de todos tus datos para que puedas comenzar a operar con Commet en producción lo antes posible.

**Opción B: Probar con datos reales en Sandbox**
Ve a la sección de API Keys y comienza a integrar el SDK para probar Commet sandbox con datos reales. Consulta nuestra [guía de inicio rápido](/library/quickstart) para ponerte en marcha en menos de 5 minutos. Esto te permite probar toda la funcionalidad usando tus propios datos antes de pasar a producción.

</Step>
</Steps>
