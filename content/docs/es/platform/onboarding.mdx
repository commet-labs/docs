En esta guía encontrarás los pasos a seguir si es la primera vez que ingresas a Commet y deseas probar sus funcionalidades base.

<Callout type="info">
Recuerda que para empezar a operar con Commet de manera real, deberás hablar con nuestro equipo para poder completar el onboarding.
</Callout>

<Steps>
<Step>

### 1. Crea una cuenta

Ve a https://sandbox.commet.co. Si es la primera vez que accedes, tendrás que completar algunos datos básicos sobre tu empresa.

</Step>

<Step>

### 2. Crea tus productos

Desde la pestaña de productos, crea tus productos, ya sean de una vez, recurrentes, fijos o vinculados a alguna métrica creada anteriormente.

<Callout type="info">
Tal vez tus productos sean por consumo o en base a seats. Podrás crear estas métricas ahí mismo. Más adelante, podrás enviar eventos vía API para probar con datos reales.
</Callout>

</Step>

<Step>

### 3. Crea una suscripción

Una suscripción es donde se reúnen todas las condiciones acordadas con tu cliente. Cada una puede tener diferentes fases y determinará Qué y Cómo cobraremos a nuestros clientes.

<Callout type="info">
Commet también está pensado para ser el centro de la verdad de tus clientes. Por ende si no tienes clientes creados tendrás que crear uno en ese momento para asociar dicha suscripción.
</Callout>

</Step>

<Step>

### 4. Simulación

En esta etapa, podrás simular tu suscripción para ver exactamente cómo se vería. Cuando llegue la fecha de facturación establecida en la suscripción, Commet enviara el invocice con el detalle de uso y cobrará al cliente.

<Callout type="warning">
En sandbox no podrás crear una suscripción real.
</Callout>

</Step>

<Step>

### 5. Próximos pasos

En este punto, tienes dos opciones:

**Opción A: Obtener una cuenta de producción**
[Reserva un slot con nuestro equipo](https://cal.com/guido-irigoyen/onboarding-meeting) para que podamos ayudarte a encontrar el mejor plan para ti. Nuestro equipo se encargará de la migración de todos tus datos para que puedas comenzar a operar con Commet en producción lo antes posible.

**Opción B: Probar con datos reales en Sandbox**
Ve a la sección de API Keys y comienza a integrar el SDK para probar Commet sandbox con datos reales. Consulta nuestra [guía de inicio rápido](/library/quickstart) para ponerte en marcha en menos de 5 minutos. Esto te permite probar toda la funcionalidad usando tus propios datos antes de pasar a producción.

</Step>
</Steps>
