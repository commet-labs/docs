# Quick Start

Get your billing system up and running in under 5 minutes.

<Callout type="info">
**Prerequisites:** You'll need a Commet account and API key. [Sign up here](https://demo.commet.co) if you don't have one.
</Callout>

<Steps>
<Step>

## Install the SDK

<Tabs items={['npm', 'pnpm', 'yarn']}>
<Tab value="npm">
```bash
npm install commet
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add commet
```
</Tab>
<Tab value="yarn">
```bash
yarn add commet
```
</Tab>
</Tabs>

</Step>

<Step>

## Initialize the client

Create your Commet client with your API key:

```typescript
import { Commet } from 'commet'

const commet = new Commet({
  apiKey: 'your-api-key',
  // Dev mode automatically enabled in development
})
```

<Callout type="warn">
**Keep your API key secure:** Never expose your API key in client-side code. Use environment variables.
</Callout>

</Step>

<Step>

## Send your first usage event

Track usage for billing in one line:

```typescript
// Track API calls for usage-based billing
await commet.usage.create({
  eventType: 'api_call',
  customerId: 'cus_A1b2C3d4E5f6G7h8I9j0K1l2',
  quantity: 1,
  timestamp: new Date().toISOString(),
})
```

</Step>

<Step>

## Manage seat licenses

Add or remove seats for subscription billing:

```typescript
// Add 5 admin seats
await commet.seats.add({
  customerId: 'cus_A1b2C3d4E5f6G7h8I9j0K1l2',
  seatType: 'admin',
  quantity: 5,
})

// Remove 2 seats
await commet.seats.remove({
  customerId: 'cus_A1b2C3d4E5f6G7h8I9j0K1l2', 
  seatType: 'admin',
  quantity: 2,
})
```

</Step>
</Steps>
